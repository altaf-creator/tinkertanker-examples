Daily Temperature Tracker

About

A graph that shows the history of daily temperature with a 5 minute interval in Singapore from as early as 2017. It is built in HTML, CSS and JavaScript. The user is able to select a data point(location) as stated in the API and select a date. The user will be able to view the graph after entering the details.
